Getting Started with Old School

Old School is a flexible, beautiful magazine-style WordPress theme with beautiful fonts, CSS3 image styles, post thumbnail images, easy to use ad spots and more!

Features

Old School offers some great easy-to-use features:

  • Homepage Featured Images

    Set your featured category and add featured images to the posts inside that category to set up your three featured images on the homepage. You MUST add Featured Images to each post inside your featured category.

  • Single Post Lead-In Image

    In addition to the homepage slider image, you can also reference a lead-in image for each post.

  • Built-in Ad Spots

    This theme allows for drop-in ad HTML viewable in the following locations: above content, below content, above sidebar, and above footer.

Installing Old School

  1. Download theme ZIP file and unzip to a local folder.
  2. Upload theme folder inside "/wp-content/themes/" folder.

Setting up Featured Post Category

Simply go to the Old School Options page and select the category you want to assign as the featured slider category.

Adding Images to Posts

When creating posts, simply enter the location of an image file for the Feature Slider area or the Lead-In image and they will be automatically cropped to fit the appropriate size, no manual resizing necessary. Keep in mind images should be at least 980px wide and 240px tall for homepage images and 480px wide and 240px tall for lead-in images.
